- **Film Slug:** Identifier used in file naming
- **Language:** Language code (en, fr, es, etc.)

- **Runtime:** Duration in minutes (cross-referenced with film_versions.json)
- **Download URL:** OpenSubtitles file ID

- **Notes:** Additional version info or quality observations
